
# Telegram Autoforwarder

Telegram Autoforwarder သည် သတ်မှတ်ထားသော သော့ချက်စာလုံးများပေါ်မူတည်၍ မက်ဆေ့ချ်များကို chat (အုပ်စု သို့မဟုတ် ချန်နယ်) တစ်ခုမှ အခြားတစ်ခုသို့ ပြန်လည်ပို့ဆောင်ပေးသည့် Python script တစ်ခုဖြစ်သည်။ ၎င်းသည် အုပ်စုများနှင့် ချန်နယ်များ နှစ်မျိုးလုံးတွင် အလုပ်လုပ်ပြီး မက်ဆေ့ချ်များကို ဝင်ရောက်ကြည့်ရှုရန် လိုအပ်သော ခွင့်ပြုချက်များသာ လိုအပ်သည်။

## အင်္ဂါရပ်များ

- သတ်မှတ်ထားသော သော့ချက်စာလုံးများပါဝင်သည့် မက်ဆေ့ချ်များကို chat တစ်ခုမှ အခြားတစ်ခုသို့ ပြန်လည်ပို့ဆောင်ခြင်း။
- အုပ်စုများနှင့် ချန်နယ်များ နှစ်မျိုးလုံးတွင် အလုပ်လုပ်ခြင်း။
- ရိုးရှင်းသော စနစ်ထည့်သွင်းမှုနှင့် အသုံးပြုမှု။

## အလုပ်လုပ်ပုံ

Script သည် Telethon library ကို အသုံးပြု၍ Telegram API နှင့် အပြန်အလှန်ဆက်သွယ်သည်။ သင်သည် သင်၏ Telegram API ID၊ API hash နှင့် ဖုန်းနံပါတ်တို့ကို အထောက်အထားဖော်ပြရန်အတွက် script သို့ ဖြည့်သွင်းပေးရသည်။ ထို့နောက် သင်ပါဝင်နေသော chat များစာရင်းကို ကြည့်ရှုရန် နှင့် မက်ဆေ့ချ်များပြန်လည်ပို့ဆောင်ရန် အသုံးပြုလိုသော chat များကို ရွေးချယ်နိုင်သည်။ ပြင်ဆင်သတ်မှတ်ပြီးသည်နှင့် script သည် သတ်မှတ်ထားသော အရင်းအမြစ် chat တွင် မက်ဆေ့ချ်အသစ်များရှိမရှိကို ဆက်တိုက်စစ်ဆေးပြီး ၎င်းတို့တွင် သတ်မှတ်ထားသော သော့ချက်စာလုံးများ တစ်ခုခုပါပါက ထိုမက်ဆေ့ချ်များကို လက်ခံမည့် chat သို့ ပြန်လည်ပို့ဆောင်ပေးသည်။

## သော့ချက်စာလုံးများ

သင်သည် သော့ချက်စာလုံးတစ်ခု သို့မဟုတ် တစ်ခုထက်ပိုကို သတ်မှတ်နိုင်သည်။ မက်ဆေ့ချ်တစ်ခုတွင် ယင်းသော့ချက်စာလုံးများ တစ်ခုခုတွေ့ရှိပါက ပြန်လည်ပို့ဆောင်ခြင်းကို စတင်စေသည်။ သော့ချက်စာလုံးများသည် case-insensitive (စာလုံးအကြီး၊ အသေးမခွဲခြား) ဖြစ်ပြီး စနစ်ထည့်သွင်းစဉ်တွင် သတ်မှတ်နိုင်သည်။

## စနစ်ထည့်သွင်းခြင်းနှင့် အသုံးပြုခြင်း

1. Repository ကို clone လုပ်ပါ

   ```bash
   git clone https://github.com/redianmarku/Telegram-Autoforwarder.git
   cd Telegram-Autoforwarder
   ```

2. လိုအပ်သော dependencies များကို ထည့်သွင်းပါ

   ```bash
   pip install -r requirements.txt
   ```

3. Script ကို ပြင်ဆင်သတ်မှတ်ပါ

   - `TelegramForwarder.py` ဖိုင်ကိုဖွင့်ပြီး သင့် Telegram API ID၊ API hash နှင့် ဖုန်းနံပါတ်တို့ကို သင့်လျော်သော variable များတွင် ဖြည့်သွင်းပါ။
   - အခြားသော ဆက်တင်များကို script အတွင်းတွင် တိုက်ရိုက်ပြုပြင်နိုင်သည်။

4. Script ကို run ပါ

   ```bash
   python TelegramForwarder.py
   ```

5. ရွေးစရာတစ်ခုကို ရွေးပါ
   - List Chats: သင်ပါဝင်နေသော chat များစာရင်းကို ကြည့်ရှုပြီး မက်ဆေ့ချ်ပြန်လည်ပို့ဆောင်ရန် အသုံးပြုမည့် chat များကို ရွေးချယ်ပါ။
   - Forward Messages: အရင်းအမြစ် chat ID၊ လက်ခံမည့် chat ID နှင့် သော့ချက်စာလုံးများကို ရိုက်ထည့်ပြီး မက်ဆေ့ချ်များပြန်လည်ပို့ဆောင်ခြင်းကို စတင်ပါ။

## မှတ်စုများ

- သင်၏ API အထောက်အထားများကို လုံခြုံစွာထားရှိပြီး အများသို့ မျှဝေခြင်းမပြုရန် သတိရပါ။
- သင်အသုံးပြုလိုသော chat များရှိ မက်ဆေ့ချ်များကို ဝင်ရောက်ကြည့်ရှုရန် လိုအပ်သော ခွင့်ပြုချက်များ သင့်တွင်ရှိကြောင်း သေချာပါစေ။
- သင့်လိုအပ်ချက်အရ script ၏ အပြုအမူနှင့် ဆက်တင်များကို ချိန်ညှိပါ။
